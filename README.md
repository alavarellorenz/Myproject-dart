# Calcolatrice Dart CLI

Una semplice applicazione di calcolatrice interattiva scritta in **Dart** (senza Flutter) che viene eseguita direttamente dal terminale.

## Funzionalit√†
* **Operazioni Base**: Somma, Sottrazione, Moltiplicazione e Divisione.
* **Menu Interattivo**: Navigazione semplice tramite numeri.
* **Gestione Errori**: Controllo per input non numerici e protezione contro la divisione per zero.
* **Ciclo Continuo**: Esegui tutti i calcoli che vuoi senza riavviare l'app.

## Requisiti
* [Dart SDK](https://dart.dev/get-dart) installato sul tuo sistema.

## Come installare ed eseguire
1. Assicurati di avere il file `app.dart` nella tua cartella.
2. Apri il terminale o il prompt dei comandi.
3. Naviga nella cartella del progetto.
4. Esegui il programma con il comando:
   ```bash
   dart run
